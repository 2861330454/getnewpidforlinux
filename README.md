检测Linux内的新增进程
下载或克隆该项目


 ```
git clone https://github.com/2861330454/getnewpidforlinux.git
 ```
安装依赖库和运行python程序
 ```
 pip install platform psutil
 python getnewpidforlinux.git
 ```
